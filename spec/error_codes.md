### System errors(1000):

- 1001 - Invalid file name.
- 1002 - Stream read failed.
- 1003 - Stream write failed.
- 1004 - Stream shutdown failed.
- 1005 - File read failed.

### HTTP errors(2000):

- 2001 - Invalid request.
- 2002 - Invalid request method.
- 2003 - Invalid status code.
